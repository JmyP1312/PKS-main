# Контрольная работа 5

## Интерактивная система управления производством

ASP.NET Core MVC приложение для планирования заказов, управления производственными линиями, учёта материалов и продукции.

## Реализовано

- База данных SQLite через Entity Framework Core.
- Сущности: продукты, материалы, производственные линии, состав продукта, производственные заказы.
- API для материалов, продуктов, линий, заказов и расчёта времени производства.
- Проверка материалов при создании заказа.
- Расчёт срока выполнения по формуле:

```text
Время = (Количество * ВремяНаЕдиницу) / КоэффициентЭффективности
```

- Панель материалов с фильтром низкого запаса и пополнением склада.
- Панель продуктов с фильтром по категории, поиском и привязкой материалов.
- Панель заказов с созданием, назначением линии, запуском, отменой, деталями и ручным обновлением прогресса.
- Панель линий со статусом, текущим заказом, прогрессом, коэффициентом эффективности и расписанием.
- Фоновый сервис для автоматического увеличения прогресса заказов на линиях с включенным авто-режимом.

## Запуск

Из папки `sem2/kr/kr5`:

```bash
dotnet restore kr5.csproj
dotnet run --project kr5.csproj
```

Если порт `5000` занят:

```bash
dotnet run --project kr5.csproj --no-launch-profile --urls http://127.0.0.1:5127
```

После запуска открыть адрес из консоли или `http://127.0.0.1:5127`.
